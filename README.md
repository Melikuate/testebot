## ⚙️ Configuration

Avant de déployer, assure-toi de configurer :  

- `TOKEN` — Token du bot Discord/plateforme  
- `PREFIX` — Préfixe des commandes (ex: `!`)  
- Variables API supplémentaires selon tes besoins
